# faculdade-segundo-semestre

Este repositório contém exercícios e projetos desenvolvidos durante meu segundo semestre na faculdade.

## Estrutura
- **doo/** → Projetos de Desenvolvimento Orientado a Objetos (POO), desenvolvidos principalmente no IntelliJ.
- **database/** → Scripts SQL e exercícios utilizando PostgreSQL.
- **java-gui/** → Sistemas com interface gráfica desenvolvidos em Java, desenvolvidos no NetBeans.

## Tecnologias
- Java
- IntelliJ
- NetBeans
- PostgreSQL
- SQL
- Swing / JavaFX

## Como rodar

## Projetos Java
Para compilar e executar um arquivo `.java`:
```bash
javac NomeDoArquivo.java
java NomeDoArquivo
```
Ou abrir diretamente no BlueJ ou outra IDE (Eclipse, IntelliJ).

## Banco de Dados (PostgreSQL)
**1.** Criar um banco de dados

**2.** Executar os scripts .sql presentes na pasta database/

**3.** Utilizar ferramentas como pgAdmin ou terminal
